# food_recommendation_service
제주도 런케이션 내에서 만들고싶은 개인프로젝트

네이버 지도나 구글 지도에 '맛집'을 검색하면 식당의 리뷰 개수,별점수 등의 정보는 있지만,
한번에 내가 원하는( 별점 높은데 리뷰개수 많은곳, 한식양식중식일식, 식당의 분위기가 어떤지, 술집인지 밥집인지 ) 곳을 찾기가 힘들다.
또한 떠오른 식당들을 일일이 클릭하여 리뷰를 확인해야한다는 불편함도 또한 존재한다.
ChatGPT처럼 LLM에 검색하면 좋겠지만, 그런 기능은 지도에서 제공해주지 않는다.

이러한 불편함이 있다는 것을 인식하였고, 이를 해결해주는 서비스를 만들어보고자 이번 프로젝트를 시작하였다.


사용자가 자신의 주소와 원하는 반경을 입력하면(도로명주소), 주소를 기준으로 반경 Nkm 의 맛집을 소개하는 서비스를 만든다.

GoogleMapsAPI를 이용하고, HTML/CSS/JS를 이용해 프론트엔드, Python의 Scikit-learn으로 데이터 머신러닝을 시킨다.
